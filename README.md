# Python To-Do List App

A simple command-line to-do list application built with Python.

## Overview

This app lets you manage a personal task list directly from your terminal. You can view, add, and delete tasks through an interactive menu — no external libraries or setup required.

## Features

- View all current tasks in a numbered list
- Add new tasks by typing them in
- Delete tasks by number with built-in error handling
- Clean menu-driven interface

## Requirements

- Python 3.x

## Getting Started

Clone the repository and run the app:

```bash
git clone https://github.com/prellwitzdarian/python-to-do-app-assignment-.git
cd python-to-do-app-assignment-
python ToDo.py
Usage
When you run the app, you'll see this menu:

Welcome to your To-Do List!
 ===MENU===
1. View Tasks
2. Add Tasks
3. Delete Tasks
4. Quit
1 — View Tasks: Displays all tasks with their numbers.
2 — Add Task: Prompts you to type a new task and adds it to the list.
3 — Delete Task: Shows the current list and asks which task number to remove.
4 — Quit: Exits the application.
